# GitHub Pages Setup for Privacy Policy

## Quick Setup Steps

1. **Push your code to GitHub** (if not already done)

2. **Enable GitHub Pages**:
   - Go to your repository on GitHub
   - Click **Settings** → **Pages**
   - Under "Source", select:
     - Branch: `main` (or `master`)
     - Folder: `/docs`
   - Click **Save**

3. **Wait 1-2 minutes** for GitHub to deploy

4. **Access your pages**:
   - Privacy Policy: `https://[your-username].github.io/[repository-name]/`
   - Delete Account: `https://[your-username].github.io/[repository-name]/lunaria/delete-account.html`
   - Example: `https://louislam09.github.io/lunaria/` and `https://louislam09.github.io/lunaria/lunaria/delete-account.html`

## Files Created

- `docs/index.html` - The Privacy Policy webpage (bilingual, English/Spanish)
- `docs/lunaria/delete-account.html` - Delete Account page (bilingual, English/Spanish)
- `docs/README.md` - Detailed setup instructions
